# BI Dashboard Web

Projeto de plataforma de BI customizada para visualização e análise de dados
corporativos, com autenticação Microsoft e dashboards interativos.

## Objetivo
Centralizar indicadores operacionais e gerenciais em um ambiente web,
com atualização automática e manual.

## Tecnologias (planejadas)
- Backend: Python + FastAPI
- Frontend: React
- Autenticação: Microsoft (Azure AD / Entra ID)
- Hospedagem: Azure
